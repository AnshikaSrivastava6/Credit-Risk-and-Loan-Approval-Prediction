# 🏦 Loan Approval / Credit Risk Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a loan application will be **approved or rejected** based on applicant financial history and other features.

It also handles **class imbalance** using SMOTE and applies multiple machine learning models including classical and neural-inspired algorithms.

---

## 🎯 Objective
To build a robust classification system that can:
- Predict loan approval status
- Handle imbalanced datasets effectively
- Compare multiple ML algorithms for best performance

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas & NumPy
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Matplotlib & Seaborn
- Jupyter Notebook
- streamlit
- joblib

---

## 📊 Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling using StandardScaler
- Train-test split

---

## ⚖️ Handling Imbalanced Data
- Applied **SMOTE (Synthetic Minority Over-sampling Technique)**
- Balanced dataset to improve model performance on minority class

---

## 🤖 Models Used
- Perceptron
- Adaline (Adaptive Linear Neuron)
- logistic regression
- support vector machine

---

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Precision & Recall
- F1 Score
- roc-auc curve

---

## 📊 Workflow
